Google fonts

Google Fonts (formerly known as Google Web Fonts) is a computer font and web font service owned by Google. This includes free and open source font families, an interactive web directory for browsing the library, and APIs for using the fonts via CSS and Android. Popular fonts in the Google Fonts library include Roboto, Open Sans, Lato, Oswald, Montserrat, and Source Sans Pro.

History

Google Fonts was launched in 2010 and revamped in 2011, 2016, and 2020. On March 3, 2020, Google updated the catalog website with support for variable fonts.[citation needed] On March 2, 2021, the Google Fonts team announced they were adding support for open source icons.
