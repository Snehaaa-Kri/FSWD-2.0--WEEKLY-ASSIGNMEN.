Q2. What are HTML Entities? List out 5 commonly used HTML entities.

Answer- An HTML entity is a piece of text ("string") that begins with an ampersand ( & ) and ends with a semicolon ( ; ).
        
        5 commonly used HTML entities are-
             a. Copyright symbol -- &copy;
             b. Greater than -- &gt;
             c. Less than -- &lt;
             d. Trademark sign -- &trade;
             e. Registered sign -- &reg;  



Q3. What is web accessibility? List some of tech assistive devices which play a major role in providing accessibility.

Answer- Web accessibility means that websites, tools, and technologies designed in such a way that people with disabilities can perceive, understand, interact and contribute to the web.

Some of the tech assistive devices are-
screen readers, screen magnification softwares, text readers, speech input softwares, etc.

Q5. Write a short note on tabindex.
     Defination: tabindex attribute specifies the tab order of an element when the "tab" button is used for nevigation. It is a global attribute and can be used on any element. It is used to make HTML elements focusable, allow or prevent them from being sequentially focusable.

     It accepts an integer as a value, with different results depending on the integer's value:
       
       1. A negative integer - (the exact value doesn't matter) means that the element is not reachble via sequential keyboardnavigation.

       2. zero value - means that the element should be focusable in sequential keyboard nevigation, after any positive tableindex value.

       3. A positive interger - means the element should be focusable in sequential keyboard nevigation, with its order defined by the value of the number.  





Q6. List any 5 semantic tags in HTML along with their description.

Answer- 1. Header element
              The <header> element represents a container for introductory content or a set of navigational links
              
              A header element typically contains:
                  > one or more heading elements (<h1> - <h6>)
                  > logo or icon
                  > authorship information
        
        2. Nevigation element
              The <nav> element defines a set of navigation links. 

        3. Aside element
              The <aside> element defines some content aside from the content it is placed in (like a sidebar).

        4. Aside tags
              The <article> element specifies independent, self-contained content.

              Article element can be used:
              > Forum posts
              > Blog posts
              > User comments
              > Product cards
              > Newspaper articles


        5. Footer element
              The <footer> element defines a footer for a document or section.       

              A <footer> element typically contains:
                  > authorship information
                  > copyright information
                  > contact information
                  > sitemap
                  > back to top links
                  > related documents
